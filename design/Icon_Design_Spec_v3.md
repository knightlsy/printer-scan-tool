# SCAN.GATE 图标设计规范 · Liquid Glass v3

> 与 v4 界面完全对齐的图标系统：冷蓝白磨砂玻璃、克制蓝点睛、像素级打磨。

---

## 一、设计语言总览

### 1.1 视觉基因（与 UI 同源）

| 维度 | UI 界面 | 图标映射 |
|------|---------|----------|
| 背景色 | `--bg-1: #eef2f9` → `--bg-3: #d8e2f2` 冷灰蓝渐变 | 瓷砖底座使用同一组渐变 |
| 玻璃材质 | `rgba(255,255,255,0.55)` + `blur(22px)` + 1px 白色高光 | 白色半透明扫描页 + 顶部高光描边 |
| 主色调 | `--accent: #2f6bff` / hi `#5689ff` / press `#1f54e0` | 取景括号 / 光束 / 品牌光点 |
| 浮动光斑 | `.blob-a/b` 径向模糊冷蓝 | 瓷砖内同色系软径向渐变 |
| 圆角体系 | `--radius-md: 16px` ~ `--radius-lg: 22px` | 方角瓷砖 rx ≈ 20%（squircle） |
| 字体气质 | Segoe UI / PingFang SC，轻标题清正文 | 图标无文字；品牌名「SCAN.GATE」在 UI 标题栏 |

### 1.2 设计哲学（Liquid Glass）

> 光穿过半透明材质。边缘有细微的高光，体内透出底层景深。主色仅作点睛绝不泛滥。每一处圆角弧度、每层透明叠加、每段字距都经得起 400% 放大审视。

## 二、图标构成解析

### 2.1 主图标（应用图标 · scan_gate_icon_v3.ico）

**尺寸规格**：多分辨率 Windows ICO — 256×256 / 48×48 / 32×32 / 16×16

**图层结构（从底到顶）**：

```
┌─────────────────────────────────────┐
│  Layer 0: 外阴影                    │  feDropShadow dx=0 dy=16 σ=20 α=22%
│    ┌───────────────────┐            │
│    │  Layer 1: 冷渐变底座│           │  linear #eef2f9 → #d8e2f2 (155°)
│    │  ┌─────────────┐  │            │
│    │  │Layer 2: 光斑A│  │            │  radial #78aaff@55% → transparent (左上)
│    │  │Layer 3: 光斑B│  │            │  radial #b4c8ff@50% → transparent (右下)
│    │  │Layer 4: 提亮  │  │            │  radial white@95% → transparent (左上)
│    │  │              │  │            │
│    │  │Layer 5: 扫描页│  │            │  white@96→82%, rx=22, feDropShadow
│    │  │  ├─ 文本行 ×4│  │            │  #c3cfe4 灰条
│    │  │  ├─ 括号框   │  │            │  accent 渐变 stroke 9px 圆头
│    │  │  ├─ 光束     │  │            │  accent bar + 辉光滤镜
│    │  │  └─ 光点     │  │            │  orb radial + dropShadow glow
│    │  │              │  │            │
│    │  └──────────────┘  │            │
│    └───────────────────┘            │
│  Layer 6: 白描边 + 蓝微描边          │  stroke white@70% + #2f6bff@10%
└─────────────────────────────────────┘
```

**语义解读**：
- **方角瓷砖** = 整个应用窗口的玻璃面板隐喻（用户打开一个「玻璃门」进入工具）
- **白色扫描页** = 共享目录中的 PDF/文档（核心操作对象）
- **取景括号** = 扫描/预览/选择动作（对应文件浏览功能）
- **蓝色光束** = 数据传输中（上传/下载）
- **发光光点（GATE）** = 品牌「门/门户」——信号节点、连接枢纽、安全网关

### 2.2 小尺寸优化策略

| 尺寸范围 | 使用字形 | 适配措施 |
|----------|---------|----------|
| ≥ 64px | 完整版 SVG | 全部细节保留 |
| 32–48px | 简化版 SVG (`glyph_small`) | 移除文本行、加粗括号(stroke 18→9)、放大光束和光点 |
| ≤ 16px | 简化版 SVG | 仅保留轮廓形状（括号+光束+光点融合为整体） |

## 三、衍生图标

### 3.1 关于对话框品牌 Logo（scan_gate_logo.png / _512.png）

- **内容**：纯品牌光点 + 光晕，透明背景
- **用途**：关于对话框中的品牌标识、启动画面水印
- **构造**：90px 半径径向渐变球体(#7ea7ff→#2f6bff→#1f54e0) + 150px 高斯模糊光晕(rgba(47,107,255,35)) + 左上白色高光亮点

### 3.2 Favicon（favicon.ico）

- **内容**：简化字形的多分辨率 ICO（64/48/32/16）
- **用途**：浏览器标签页图标（本地 webview 内页）、快捷方式
- **说明**：基于 glyph_small 版本渲染，保证小尺寸可读性

### 3.3 单色版（scan_gate_icon_mono.png）

- **内容**：256px 单色高对比度 PNG
- **规则**：
  - 原 accent 蓝区域 → 强化为 `#2f6bff`（alpha boost）
  - 浅色区域（lum > 200）→ 纯白
  - 其余区域 → `#2f6bpp` @65% alpha
- **用途**：深色任务栏反转模式、高对比度辅助功能、打印输出占位符

## 六、使用指南

### 替换项目图标

将 `assets/icon/scan_gate_icon_v3.ico` 复制到项目根目录并重命名为 `scan_gate_icon.ico`：

```bash
cp assets/icon/scan_gate_icon_v3.ico ../scan_gate_icon.ico
```

然后重新运行 PyInstaller 打包（`--icon "scan_gate_icon.ico"`），新图标即生效。

### 在关于对话框中使用 Logo

可将 `scan_gate_logo.png` 作为 `<img>` 引入 `index.html` 的关于弹窗，替代纯文字的品牌展示。

### 未来扩展方向

- **状态变体**：连接态（绿光环）、断开态（红/灰光环）、传输中（脉冲动画）
- **macOS .icns**：从同一 SVG 导出，调整圆角为 Apple squircle
- **Windows 11 Mica 适配**：若未来迁移到 WinUI 3，图标已预留 Fluent Design 兼容性（方角 + 亚克力 + 微光）

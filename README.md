# 打印机扫描工具 - 带文件管理UI的应用程序

## 概述
这是一个将原有的批处理脚本 `打印机扫描.bat` 转换为图形界面应用程序的工具。该工具提供了完整的文件管理UI界面，可以方便地连接网络共享、浏览文件、上传下载文件等。

## 功能特性

### 1. 网络共享管理
- **连接共享**: 输入共享路径、用户名和密码连接网络共享
- **断开连接**: 断开已连接的共享
- **测试连接**: 测试网络连通性
- **打开文件夹**: 在资源管理器中打开共享目录

### 2. 文件管理
- **文件浏览**: 以列表形式显示共享目录中的文件和文件夹
- **文件排序**: 按名称、大小、修改时间排序
- **文件操作**:
  - 双击打开文件/进入文件夹
  - 上传文件到共享
  - 从共享下载文件
  - 删除文件/文件夹
- **实时刷新**: 随时刷新文件列表

### 3. 状态监控
- **操作日志**: 记录所有操作的详细日志
- **状态显示**: 实时显示当前状态信息

## 使用方法

### 方法一：直接运行Python脚本
1. 确保已安装Python 3.6+
2. 运行 `python 打印机扫描工具.py`

### 方法二：构建可执行文件（推荐）
1. 双击运行 `build_exe.bat`
2. 等待构建完成（约1-2分钟）
3. 构建完成后，在 `dist` 文件夹中找到 `打印机扫描工具.exe`
4. 双击运行可执行文件

### 方法三：手动构建
```bash
cd "程序所在目录"
pyinstaller --onefile --windowed --icon=NONE --name="打印机扫描工具" "打印机扫描工具.py"
```

## 界面说明

### 主界面布局
1. **配置区域**: 设置共享路径、用户名、密码
2. **控制按钮**: 连接、断开、刷新、上传、下载、删除等操作按钮
3. **文件列表**: 显示共享目录中的文件和文件夹
4. **状态区域**: 显示操作日志和状态信息

### 文件列表功能
- **双击文件**: 打开文件
- **双击文件夹**: 进入文件夹
- **右键菜单**: 支持更多操作（未来版本）
- **多选支持**: 支持多文件操作（未来版本）

## 常见问题

### Q1: 无法连接共享
- 检查网络连接是否正常
- 确认共享路径、用户名、密码正确
- 检查防火墙设置
- 确保有访问权限

### Q2: 文件列表为空
- 确认已成功连接共享
- 检查共享目录中是否有文件
- 尝试刷新文件列表

### Q3: 无法上传/下载文件
- 检查是否有写入权限
- 确认目标路径存在
- 检查磁盘空间是否充足

### Q4: 程序运行缓慢
- 关闭不必要的后台程序
- 检查网络连接速度
- 减少同时操作的文件数量
